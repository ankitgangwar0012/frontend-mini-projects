## ‚ú® Project Features

- **Complete Periodic Table (118 Elements)**  
  All chemical elements arranged in correct order and structure.

- **Category-Based Color Coding**  
  Different colors for:
  - Alkali Metals  
  - Alkaline Earth Metals  
  - Transition Metals  
  - Basic Metals  
  - Metalloids  
  - Nonmetals  
  - Halogens  
  - Noble Gases  
  - Lanthanides  
  - Actinides  

- **CSS Grid Powered Layout**  
  Accurate placement using `grid-template-columns` and custom empty grid blocks.

- **Hover Animation on Elements**  
  Smooth scaling animation for interactive feedback.

- **Element Information Display**  
  Each element shows:
  - Atomic Number  
  - Symbol  
  - Name  
  - Atomic Weight  

- **Legend Section Included**  
  Helps users easily understand element categories.

- **No JavaScript**  
  100% static project using only HTML & CSS.

- **Clean and Readable Code**  
  Beginner-friendly and well-structured.
